# Smart Resume Analyzer & Job Matcher

## 📌 Overview
Smart Resume Analyzer is an AI-based web application that evaluates resumes by generating an ATS score, identifying missing skills, and matching candidates to job roles. It uses NLP techniques to extract relevant information and provide actionable suggestions for improvement.

---

## 🚀 Features
- 📄 Resume upload (PDF/DOCX)
- 📊 ATS score generation (0–100)
- 🧠 Skill extraction using NLP
- 🎯 Job-role matching with percentage score
- ⚠️ Missing skill detection
- 💡 Improvement suggestions dashboard

---

## 🛠️ Tech Stack
- Frontend: React.js  
- Backend: FastAPI / Flask  
- Language: Python  
- Database: SQLite / MongoDB
